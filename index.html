Description: This is a 3d renderer built in Snap! that allows you to import any OBJ File and see it as a wireframe. Features: It can import any OBJ File into the program and convert it to vertex coordinates and a list of edges. It determines where each of the vertices go by converting the 3d vertex coordinates to 2d coordinates in the display in Snap! with projection. It then draws lines between the listed vertices in each edge. Directions: Find any obj file of your choosing, if you don't want to find one, the link will provide you with one. Then once the 3d model loads, you can rotate the model around the y axis by holding left click and moving your mouse, and you can also zoom in and out with the scroll wheel.
